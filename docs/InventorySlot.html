Class: InventorySlot

InventorySlot(x, y, image, game)

Class representing an individual slot that will be placed within the Inventory window.

Constructor

new InventorySlot(x, y, image, game)

Create an image to show the boundary of the slot as well as a sprite to show the content.
Parameters:
Name Type Description
x number The x coordinate of the slot relative to the camera.
y number The y coordinate of the slot relative to the camera.
image string The name of the image showing the boundaries.
game Phaser.Scene The Scene the slot should be created in.

mouseClick(mouseX, mouseY, inHand) → {object}

Called when mouse is clicked. Checks if the mouse has clicked within the boundaries of the slot and, if it has, swaps the item in the player's 'hand' with the item in the slot.
Parameters:
Name Type Description
mouseX number The X coordinate the mouse clicked on.
mouseY number The Y coordinate the mouse clicked on.
inHand object Object representation of the item in the player's 'hand'.
