Class: FloorManager

FloorManager(boundary_defs, player, inventory, game)

Class that manages changing the floor image, boundaries and items for the floor the player is on as well as allowing them to move up and down via stairs.

Constructor

new FloorManager(boundary_defs, player, inventory, game)

Creates floor image, Barriers object and DroppedItemHandlers and sets everything up for floor 0.
Parameters:
Name Type Description
boundary_defs Array Array of arrays of boundary definitions for each floor
player Player The Player object, needed for coordinates to check if on stairs
inventory Inventory The Inventory object, needed to create DroppedItemHandlers
game Phaser.Scene The Scene the player is in.
